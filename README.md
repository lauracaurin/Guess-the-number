# Juego de Adivinar el Número 🎲

Este es un simple juego de adivinanza en el que debes adivinar un número aleatorio entre 1 y 100. El juego te proporciona pistas sobre si el número que ingresaste es demasiado alto o bajo.

## Funcionalidades 🕹️

- **Adivina el Número**: Introduce un número entre 1 y 100.
- **Pistas**: El juego te dice si tu número es demasiado alto, bajo, o correcto.
- **Contador de Intentos**: Muestra cuántos intentos has hecho.
- **Reiniciar**: Puedes reiniciar el juego para intentar nuevamente.

## Cómo Jugar 🎯

1. **Introduce un Número**: Escribe tu número en el campo de entrada.
2. **Haz clic en "Adivinar"**: Presiona el botón para ver si adivinaste el número correcto.
3. **Recibe Pistas**: El juego te dirá si el número es demasiado alto o bajo, o si adivinaste correctamente.
4. **Reinicia el Juego**: Usa el botón de reinicio para comenzar una nueva partida.

## Tecnologías Utilizadas 🛠️

- **HTML**: Estructura del juego.
- **CSS**: Estilos para el diseño.
- **JavaScript**: Lógica del juego y manejo de eventos.
